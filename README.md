AttendanceSorter is a lightweight, browser-based tool I built to help sort student names alphabetically for attendance sheets — fast.

⚠️ Disclaimer: This is NOT an official program of ICCT Colleges Foundation Inc. or the San Mateo Satellite Campus. It is an independent personal project made purely to make my life as a student secretary a little easier. It has no affiliation with, endorsement from, or connection to ICCT in any official capacity.


What It Does

Paste student names from Excel, Google Sheets, or just type them in
Sorts them A → Z instantly with letter dividers
Export as .xlsx (Excel file) with attendance header pre-filled
Copy list in tab-separated format — pastes cleanly back into Excel
Print / Save as PDF
Dark and light mode
Remembers your theme preference


How to Use

Download the .html file
Open it in any modern browser — no installation needed, no internet required after loading
Fill in Professor, Subject, Section, and Date
Paste your names → click Sort A → Z
Export, copy, or print
